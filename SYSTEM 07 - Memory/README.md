# SYSTEM 07 - Memory

**Organizational Memory and Learning**

SYSTEM 07 - Memory defines how the organization captures, maintains, and leverages its accumulated experience and learning. Organizational memory is the foundation for continuous improvement and competitive advantage.

---

## System Overview

SYSTEM 07 - Memory is the organizational learning system that defines:

1. **Organizational Memory**: What memory is and why it matters
2. **Explicit Memory**: Documented knowledge capture and organization
3. **Tacit Memory**: Transfer of unspoken, experiential knowledge
4. **Memory Systems**: Infrastructure for managing organizational memory
5. **Learning Culture**: Building a culture of continuous learning

## Key Concepts

### Memory is Competitive Advantage

Organizations that learn:
- Improve continuously
- Adapt quickly to change
- Make better decisions
- Avoid repeating mistakes
- Build institutional wisdom
- Compound improvements over time

### Memory Has Two Forms

**Explicit**: Documented, codified, searchable, shareable
**Tacit**: Experiential, judgment-based, relationship-dependent, at-risk

### Systematic Approaches Matter

- Don't rely on chance or informal processes
- Build systems and practices
- Create infrastructure
- Establish rhythms (retrospectives, reviews)
- Make memory accessible
- Maintain over time

### Culture Enables Memory

- Psychological safety enables learning
- Growth mindset enables development
- Curiosity enables exploration
- Experimentation enables discovery
- Reflection enables understanding
- Sharing enables leverage

### Memory Requires Investment

- Time for retrospectives and reflection
- Resources for knowledge systems
- Effort to document and organize
- Leadership modeling and reinforcement
- Maintenance and updates
- Continuous evolution

## Quick Reference

### Memory Types

- **Explicit**: Documented knowledge (processes, decisions, strategies)
- **Tacit**: Experiential knowledge (judgment, skills, wisdom)
- **Structural**: Embedded in systems and roles
- **Cultural**: Shared values, stories, symbols

### Memory Lifecycle

1. **Capture**: Observe, experience, document
2. **Organize**: Structure for discovery
3. **Retrieve**: Find and use
4. **Evolve**: Update with new learning

### Building Memory Systems

1. **Infrastructure**: Repository, tools, practices
2. **Ownership**: Clear responsibility
3. **Rhythm**: Regular retrospectives and reviews
4. **Culture**: Psychological safety, learning mindset
5. **Investment**: Time, resources, attention

### Learning Culture Elements

- Psychological safety (safe to fail)
- Growth mindset (can improve)
- Curiosity (why and why not?)
- Experimentation (try and learn)
- Reflection (what did we learn?)
- Sharing (help others learn)
- Accountability (expected to grow)

### Key Practices

- Weekly retrospectives
- Monthly organizational reviews
- Quarterly strategic reflection
- After-action reviews for major events
- Knowledge capture interviews
- Mentoring and coaching
- Brown bag lunches
- Story sharing
- Documentation maintenance
